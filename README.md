# JK Testing

### Requirements
- Node
- Typescript
- Co-pilot enabled

### Install instructions

1. Clone repository: `git@github.com:JarettForzano/JK-Testing.git`
2. Install dependencies `npm install`
3. Compile the extension `tsc` -> will appear inside of `out`
4. In `extension` click `Run and Debug` - Wait for new window to open up
5. Open copilot
   1. Type `@tutor` in the chat
