# JK Testing

### Install instructions

1. Clone repository: `git@github.com:JarettForzano/JK-Testing.git`
2. Find the `src/extension.ts`
3. Click `Run and Debug` - Wait for new window to open up
4. Command + Shift + P
   -  Search "Hello world"
5. Run it
