# JK Testing

### Requirements
- Node

### Install instructions

1. Clone repository: `git@github.com:JarettForzano/JK-Testing.git`
2. Install dependencies `npm install`
3. Find the `src/extension.ts`
4. Click `Run and Debug` - Wait for new window to open up
5. Command + Shift + P
   -  Search "Hello world"
6. Run it
